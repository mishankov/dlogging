# dlogging
Python logging module

# Why?
To use it my own projects and practice decorators and console output styling

# How to use?

```python
import dlogging as log


log.debug('test_log')
log.info('test_log')
log.warning('test_log')
log.error('test_log')
log.critical('test_log')
log.info_forced('test_log')
```
